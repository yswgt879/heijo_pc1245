## セットアップ

本体へ、以下の順でセットアップします。

現在、プログラム音声ファイルは公開していません。<br>
よって、プログラムファイル`heijo.bas`を、何らかの方法でプログラム音声ファイルに変換して転送するか、プログラムリストを手入力する必要があります。<br>

### 補足
`heijo_org.bas`は、`heijo.bas`にコメントの追加と、命令文等の省略を無くしたものとなります。<br>
どちらも、処理内容に違いはありません。<br>
（プログラムの可読性を高めたものです。）<br>

PC-1245の場合、メモリの都合上、途中で入らなくなります。<br>
PC-1251/55であれば入力できますが、`コメント 'の行`は削除してください。<br>

## 起動方法

`DEFキー`を押し、`=キー`を押してください。<br>

## ゲームのジャンル

穴掘りアクション<br>

## ストーリー

なんと見事な平城京（710年）。<br>
天平の都・平城京に、どう考えても時代を間違えたエイリアンが出現した。<br>

都の警備を担う衛門府が出動するが、ここで深刻な事実が判明。<br>
プレイヤーは弱い。とにかく弱い。<br>
剣も槍も重すぎて、持った瞬間に腰を痛める。<br>

残された装備は、なぜか正式支給された一本のスコップ。<br>
衛門府は結論を出す。<br>
戦は腕力ではない。穴である。<br>

プレイヤーは今日も都を守るため、掘って、誘って、落として、埋める。<br>
平城京の平和は、あなたの筋力ではなく、穴の完成度にかかっている。<br>

## ゲームのルール

プレイヤーは、平城京を守る衛門府です。<br>
`左右`で移動します。<br>

`エイリアンの向きに合わせ、隣にきたタイミング`で、`↓キー`で地面に穴を掘ります。<br>
`↑キー`で穴を埋めます。<br>
穴を掘るのも埋めるのも、あなたの仕事です。<br>

エイリアンはとても強く、直接触れると捕まります。<br>
捕まると喰べられてしまい、昇天します（ミス扱い）。<br>
人生、逃げるが勝ちです。<br>

捕まると、ゲームオーバーとなります。<br>
都の平和と、あなたの努力が報われます。<br>

## 操作方法

### 《ゲーム》

| キー           | 役割                                           |
|----------------|------------------------------------------------|
| `４`・`６キー` | 衛門府を左右に操作します。<br>
|                | 左右端を超えて移動すると、反対側へ移動します。 |
| `↓キー`       | エイリアンのいる地面に穴を掘ります。           |
| `↑キー`       | エイリアンの落ちた穴を埋めます。               |

## PB-100版『平城京エイリアン』との違い

PC-1245のメモリの都合上、かなり制限されています。<br>
具体的には、以下のとおりです。<br>

* タイトル・ゲームオーバー等のメッセージが無い。
* 残機は無く、一人のみ。
* 穴掘りの過程は無く、１回のみの穴掘り・穴埋めとなる。<br>
よって、戦略性は無く、純粋なアクションゲームとなっている。
* プレイヤーの方向をいきなり転換可能。（向くだけの操作ができない）
* 音の種類は少しだけ。
* エイリアンの動きが単純過ぎる。
* 壁が無い。
* ステージが無い。エンドレス。

等々、無いことだらけです。<br>
そのため、ゲーム名の頭へ「Tiny」を付加しました。<br>
PC-1251があれば、全て網羅＋αとしたものを、制作したいのですが・・。<br>
